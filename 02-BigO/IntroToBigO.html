Big O Notation

Big O is a way of evaluating different solution and comparing code performance. Is a way to formalize fuzzy counting that allows us to talk formally about how the runtime of an algorithm grows as the inputs grow. We care about the trends. Big O is the worst case scenario - the upperbound.
